## 一、竞品是什么（一句话）

一个**网页版（SaaS）**的微信聊天记录分析工具：你把导出的 `CSV / TXT` 上传到它网站，它一键生成一份**英文"研究报告"风格**的可视化分析报告，**按次收费 ¥19.9 / 次**。

- 品牌 / 网站：**Chat Report · wechatreport.com**
- 形态：**注册登录网站 → 上传文件 → 在线生成报告**（不是本地软件，虽然商品名写了 Mac/Windows）
- 商品页数据：**¥19.99 / 次，已售 781**

## 二、它的功能盘点（从商品详情图逐项提取）

### 可分析的关系类型
- 情侣
- 好友
- 群聊

### 报告里的具体分析模块
| 模块 | 具体图表 / 能力 |
|---|---|
| **统计分析** | 每小时聊天分布、日聊天分布、周聊天热力图、聊天时间分布图、词频统计柱状图、高频词云图 |
| **情感分析** | 情绪分布饼图/环形图、**情绪分布热力图**、**情绪波动折线图** |
| **MBTI 分析** | 聊天频率分析、情绪分布分析、回复时间分析、**MBTI 人格预测**（预测双方是 INTJ / ESFP 之类） |
| **报告形态** | 一整份英文风"Research report on WeChat records"，分 Part 1/2/3，偏"学术报告/论文"观感 |

### 它主打的营销场景（卖点话术）
微信聊天记录分析 ｜ 情侣情感报告 ｜ **Crush 关系解密** ｜ **MBTI 恋爱人格测试** ｜ **AI 情话生成器** ｜ 生日纪念日创意礼物 ｜ 异地恋专属回忆 ｜ 群聊热梗统计 ｜ 高频词云图 ｜ 情感热力图 ｜ **520 礼物**

### 产品自述的三个亮点
1. **简单**：只需上传 CSV/TXT，一键生成
2. **丰富**：模板持续更新（目前 8 个报告模板）
3. **隐私**：报告生成后"聊天数据自动删除"

### 使用流程（3 步）
1. 获取聊天数据（它提供教程 `wechatreport.com/blog/tutorial`）
2. **注册登录网站**
3. **上传** CSV/TXT → 一键生成报告

## 三、它的收费模式（这套设计值得学）

- **单价**：¥19.9 生成一次报告
- **满减券**（诱导一次多买）：
  - 满 59 减 30 → 约等于买 3 次
  - 满 119 减 80 → 约等于买 6 次
  - 满 159 减 110 → 约等于买 8 次
- **会员钩子**：买满 8 次自动成为会员，可**永久无限次使用**（"因为目前只有 8 个模板，买满 8 次相当于全都能出"）
- **转化话术**："下单秒发货"，营造即时满足

> 判断：这是一个设计过的**付费漏斗**——用 19.9 低价钩子入门，用满减把客单价从 20 元拉到 50~160 元，再用"会员永久"锁定长期用户。**已售 781** 说明"为聊天记录分析付费"这件事是真实成立的市场。

## 四、年轮 vs Chat Report 对照表

| 维度 | Chat Report（竞品） | 年轮 Nianlun（我们） |
|---|---|---|
| **形态** | 网页 SaaS，需注册登录 | 微信小程序，手机端即用 |
| **数据去向** | **上传到服务器**再生成（宣称生成后删除） | **全程本地处理，聊天记录不上传** ✅ 硬差异 |
| **收费** | ¥19.9/次，满减+会员漏斗 | 目前免费（商业模式待定） |
| **关系类型** | 情侣 / 好友 / 群聊 | 好友（群聊在候选清单） |
| **统计图表** | 时间分布、热力图、词频、词云 | 概览有月度趋势、时段热力图、词云；详情页更细 |
| **情绪分析** | 情绪饼图 + 情绪热力图 + 情绪波动折线 | **逐月情绪走势折线图**（双方情绪对比在做） |
| **MBTI 人格预测** | ✅ 预测双方 MBTI（吸睛卖点） | ❌ 暂无 |
| **AI 情话生成器** | ✅ 有（娱乐向） | ❌ 暂无 |
| **AI 好友关系/画像** | ❌ 无好友管理概念 | ✅ AI 推断关系职务、AI 好友画像（在做） |
| **关系网络图** | ❌ 无 | ✅ 以"我"为中心的气泡关系网 |
| **好友清单管理** | ❌ 无（一次只出一份报告） | ✅ 可搜索/排序/改关系/真实名字(WeLive) |
| **年度报告海报** | 偏"英文研究报告"风 | 一页式**海报**，可存相册分享 |
| **多次导入合并** | 上传单文件 | ✅ 多次导入自动累加合并 |
| **报告观感** | 学术/论文风，信息密度高 | 海报风，社交分享友好 |

### 一句话总结差异
- **年轮更强的地方**：纯本地不上传（隐私）、关系网 + 好友管理体系、小程序即开即用、多次导入合并、海报社交分享。
- **竞品更强/更会卖的地方**：MBTI 人格预测、AI 情话生成器这类**娱乐/情感话题点**；成熟的**付费漏斗**；"研究报告"式的信息密度；情侣/crush 场景的精准营销包装。

## 五、拿去问客户的问题（重点）

> 目的：借竞品验证"年轮下一步该补什么、能不能收钱、隐私是不是硬需求"

### 关于功能（对方有、我们没有的，要不要跟）
1. 竞品主打 **MBTI 人格预测**（预测双方是什么人格），这个"话题性"功能，客户觉得目标用户会喜欢吗？值不值得我们做？
2. 竞品有 **AI 情话生成器 / Crush 关系解密 / 520 礼物** 这类偏娱乐、情感的玩法——客户的定位是**工具**（认真分析）还是**礼物/社交玩具**？这决定我们要不要做这些。
3. 竞品把 **情侣 / 好友 / 群聊** 分成三种独立分析。客户最看重哪一种场景？（我们群聊还没做，要不要优先补）
4. 竞品报告是**英文"研究报告"风**、信息密度高；年轮是**海报风**、适合发朋友圈。客户更想要哪种观感？

### 关于隐私（这是我们的核心差异，务必验证）
5. 竞品需要**把聊天记录上传到它服务器**才能生成（虽然说事后删除）。客户或客户的用户，**能接受上传吗**？还是"绝不出手机"是硬需求？
   —— 如果是硬需求，那**"纯本地不上传"就是年轮最大的卖点**，要重点打。

### 关于商业模式（对方已经在赚钱）
6. 竞品 **¥19.9/次、已售 781**，说明这事能收费。客户希望年轮：
   - ⬜ 完全免费（做口碑/送礼）
   - ⬜ 按次/按报告收费（对标竞品）
   - ⬜ 会员制（一次买断/包年）
   - ⬜ 先免费获客，后续增值功能收费
7. 如果收费，客户觉得**合理定价**是多少？（竞品单次 19.9，会员相当于 ~160 元）

## 六、给我们自己的初步判断（内部参考，不一定给客户看）

1. **隐私是我们唯一的、结构性的护城河**——竞品做成 SaaS 就注定要上传，我们做成小程序本地处理就注定不用上传。这一点在营销上要放到最前面。
2. **"话题性功能"投入产出比高**：MBTI 预测、情话生成器本质是一次 AI prompt 调用，开发成本低但传播性强，是低成本获客点，可以考虑做。
3. **付费漏斗可借鉴**：竞品"低价钩子 + 满减 + 会员永久"这套很成熟，若年轮要商业化，可参考。
4. **不要盲目跟"研究报告"风**：那是竞品的差异化，年轮的海报 + 关系网是我们自己的差异化，别丢了。
