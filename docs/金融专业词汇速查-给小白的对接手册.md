# 金融专业词汇速查 · 给小白的对接手册

> 用途：本项目客户是**投资圈的最终用户**（在上海，日常和各路「首席 / 大咖」聊股票、荐股）。
> 这份手册把客户需求文档里出现的、以及你跟他对接时大概率会撞上的金融黑话，
> 用大白话讲清楚，并给出「客户可能会这么说」的例句，帮你听懂、接得住话。
> 你**不需要精通**这些（客户自己也说过：他口述当 prompt，专业术语交给 AI 落地），
> 但听懂关键词能让沟通顺很多。
>
> 阅读顺序建议：先看「一、必须先搞懂的 6 个词」，其余按需查。

## 一、必须先搞懂的 6 个词（看完这 6 个就能听懂大半）

| 词 | 一句话解释 | 客户可能会这么说 |
|---|---|---|
| **二级市场** | 已经上市的股票在股民之间买来卖去的市场（就是我们平时炒的 A 股）。**这是本项目最核心的模块。** | 「二级这块是重中之重」 |
| **一级市场** | 公司还没上市、在私底下融资的阶段（VC/PE 投的就是这个）。 | 「他是做一级的，梳理下投资方向就行」 |
| **首席** | 券商研究所的「首席分析师」，某个行业里最权威的研究员，他推的票分量重。 | 「这个票是××首席推的」 |
| **荐股 / 推票** | 「推荐一支股票」的行话。谁在什么时候推荐了哪支票，是本项目要反查的核心。 | 「谁推了什么票、逻辑是什么」 |
| **标的** | 「投资对象」的统称，在股票语境里就是指「某支股票」。**核心标的** = 很多人都在推、最值得关注的那几支。 | 「50 个核心标的」 |
| **逻辑** | 「为什么值得买」的理由。买股票不是拍脑袋，要讲得出 1、2、3 条逻辑。 | 「这个票的逻辑是 MOC 涨价」 |

## 二、市场与交易结构

| 词 | 解释 |
|---|---|
| **A 股** | 在中国大陆（上海/深圳交易所）上市、用人民币交易的股票。 |
| **二级市场** | 见上。已上市股票的流通买卖市场。 |
| **一级市场** | 未上市公司的股权融资市场（投的是「还没上市的公司」）。 |
| **一级半市场** | 介于两者之间（如定增、Pre-IPO），偶尔会听到，了解即可。 |
| **上市 / IPO** | 公司第一次公开发行股票、登陆交易所。IPO = Initial Public Offering。 |
| **未上市投资人** | 专门投一级市场（没上市的公司）的人。文档里「一级投资人」就是这类，分析他只需梳理「投资方向」。 |

## 三、人物角色（客户嘴里的「谁」）

| 词 | 解释 |
|---|---|
| **首席（首席分析师）** | 券商研究所里某行业最资深的分析师，观点最有影响力。 |
| **大咖 / 大 V** | 泛指圈内有名气、说话有分量的人。 |
| **卖方** | 券商研究所这一方 —— 做研究、写报告、把观点「卖」给机构，首席就属于卖方。 |
| **买方** | 真正拿钱买股票的机构，如公募基金、私募基金。 |
| **公募 / 私募** | 公募基金面向大众公开募集（如支付宝里买的基金）；私募面向少数有钱人，门槛高、更灵活。 |
| **游资 / 散户 / 机构** | 游资＝短线大资金；散户＝个人小股民；机构＝基金等专业大资金。 |
| **产业方 / 董事长** | 上市公司背后的实业人物。文档「产业类」好友就指这些人（看其任职公司、董事长身份等）。 |

## 四、个股信息与估值（股票模块「第一层｜票的基本盘」用到的词）

| 词 | 解释 | 备注 |
|---|---|---|
| **现价** | 股票现在的价格。本项目从「咱们的数据库」取数，不求实时。 | |
| **市值** | 公司在股市里的总价值 = 股价 × 总股数。单位常用「亿」。 | 「看到 500 亿 / 1000 亿」指的是**目标市值**。 |
| **目标市值** | 预期这家公司能涨到的总价值。客户很关心「这票能看到多少亿」。 | 从当前市值 → 目标市值，就是涨幅空间。 |
| **目标价** | 预期股价能到的价位（另一种表达涨幅空间的方式）。 | |
| **涨幅倍数** | 预计能涨几倍（如从 400 亿到 800 亿 = 看 2 倍）。本项目要能按倍数排序。 | 「看多少倍」 |
| **估值** | 判断一支股票「贵不贵」的方法体系（市值、目标价都属于估值范畴）。 | |
| **PE / 市盈率** | 最常用的估值指标：股价 ÷ 每股盈利。数字越高一般越贵。 | 听到「多少倍 PE」时指这个，别和「涨几倍」搞混。 |
| **PB / 市净率** | 另一个估值指标：股价 ÷ 每股净资产。 | 了解即可。 |

> 📌 **易混提醒**：客户说的「看多少倍」通常指**涨幅倍数**（还能涨几倍）；
> 而「多少倍 PE」是**估值贵不贵**。同样是「倍」，含义完全不同，注意区分。

## 五、研究逻辑与催化剂（股票模块「第二层｜推荐逻辑」用到的词）

| 词 | 解释 |
|---|---|
| **推荐逻辑 / 投资逻辑** | 「为什么该买」的核心理由，通常分条列（逻辑 1/2/3）。本项目要逐条沉淀。 |
| **催化剂** | 能推动股价上涨的具体事件/因素（如政策、涨价、新订单、业绩超预期）。 |
| **涨价逻辑** | 因为某产品要涨价，公司利润会增厚 → 股价上涨。文档示例「**MOC 涨价**」就是一条涨价逻辑。 |
| **基本面** | 公司自身的经营状况（营收、利润、行业地位）。「基本面好」= 公司本身经营扎实。 |
| **题材 / 概念** | 市场热点标签（如「AI 概念」「半导体概念」），短期能带动情绪和股价。 |
| **业绩 / 财报** | 公司赚不赚钱的官方成绩单，每季度公布，是逻辑兑现的关键验证。 |
| **赛道 / 板块** | 一类同行业的股票合称（如「半导体材料板块」）。 |

## 六、文档里出现的具体案例（了解背景，聊天时不发怵）

这些是客户举例时提到的真实公司，都属于**半导体材料**这个赛道，你不用懂技术，知道「它们是同一类票、被当作核心标的举例」即可：

| 名称 | 是什么 |
|---|---|
| **江化微** | 一家半导体材料上市公司。文档里作为「票的基本盘」示例。 |
| **和林微纳、国瓷材料** | 两家半导体/新材料公司。文档示例「市值 400 亿 → 800 亿，逻辑为 MOC 涨价」。 |
| **MOC** | 一种半导体材料/工艺相关的东西。你只需知道「MOC 涨价」是一条**看涨逻辑**（涨价 → 公司多赚钱 → 股价涨）。 |

> 你完全不必研究这些公司的技术，客户也没要求。记住它们「是股票、被谁推过、逻辑是什么」这层信息就够了 —— 这正是本项目要做的事。

## 七、投资风格与「好友画像」相关（AI 好友画像里的「投资偏好」用到的词）

本项目 AI 会给好友生成「投资偏好」解读，涉及这几个维度：

| 词 | 解释 |
|---|---|
| **风险偏好** | 这个人敢不敢冒险：**激进**（爱搏高收益、能扛大波动）↔ **保守 / 稳健**（求稳、怕亏）。 |
| **关注品类** | 他主要关注哪类资产/行业（股票、基金、房产，或某个板块）。 |
| **财富线索** | 从聊天里能看出的、关于他财力/资金体量的蛛丝马迹。 |
| **决策风格** | 他怎么做投资决定：靠数据研究、靠人脉消息、还是跟感觉。 |
| **配置 / 仓位** | 「配置」= 把钱分散投到不同资产；「仓位」= 某个投资占总资金的比例（「重仓」= 押了很多，「空仓」= 没买）。 |

## 八、可能听到、但和本项目关系不大的词（扫一眼即可）

| 词 | 一句话 |
|---|---|
| **多 / 空** | 「看多」= 觉得会涨；「看空」= 觉得会跌。 |
| **牛市 / 熊市** | 大盘整体上涨的行情 / 整体下跌的行情。 |
| **回撤** | 从最高点跌下来的幅度，衡量「亏了多少」。 |
| **止盈 / 止损** | 涨到目标就卖（止盈）/ 跌破底线就割肉离场（止损）。 |
| **龙头** | 某板块里最强、最有代表性的那支股票。 |
| **建仓 / 加仓 / 减仓 / 清仓** | 开始买入 / 买更多 / 卖一部分 / 全部卖光。 |

## 九、和客户对接的 3 条小建议

1. **听不懂就直接问「这支票的逻辑是什么」**——客户很习惯讲逻辑，这句话最安全、最专业。
2. **抓三层信息**：客户描述一支票时，脑子里对齐文档的三层 ——
   ① 谁推的、什么时候、看多少倍（基本盘）→ ② 为什么推（逻辑）→ ③ 背后公司谁说了什么（公司信息）。
3. **记住两个交叉视图**：客户最终想要的是「**以票查人**（这票有多少人在推）」和「**以人查票**（这个人推过哪些票）」。聊需求时始终围绕这两个方向，就不会跑偏。
