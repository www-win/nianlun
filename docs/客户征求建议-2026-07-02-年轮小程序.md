# 年轮 · 小程序体验反馈与建议征集

> 日期：2026-07-02
> 致：客户　｜　来自：开发方
> 一句话：您当初参考的小红书那款「聊天记录分析报告」，我已经把它做成了一个**微信小程序**。现在功能已经能在真机上跑起来了，想请您体验后，把感受和建议告诉我，方便我按您真正想要的方向继续打磨。

---

## 一、先对齐一下：您想要的 vs 我做出来的

您当初参考的小红书那款，是一个**网页工具**：注册登录、把聊天记录**上传到它的网站**，它生成一份分析报告，按次收费。

我做的「**年轮**」，思路一致，但在两件事上特意做了不同：

1. **做成了微信小程序**——手机上直接用，不用装电脑软件、不用注册网站。
2. **全程本地处理、聊天记录不上传**——解析、统计都在您手机里完成，聊天内容不会离开手机。这一点比原来那款更让人放心。

> 想先请您确认：**"不上传、更私密"这个方向，是您想要的吗？** 还是说您其实不介意上传、只要报告好看就行？

---

## 二、年轮现在已经能做的事（请您边体验边打分）

小程序底部 5 个板块：**导入 → 概览 → 好友 → 关系网 → 报告**。

| 功能 | 能做什么 | 您的感受（满意 / 一般 / 要改） |
|---|---|---|
| **导入聊天记录** | 支持 CSV/JSON/TXT/HTML，还能直接选 ZIP 压缩包；可多次导入自动合并 | |
| **年度概览** | 好友数、全年消息数、活跃天数、聊得最多排行、关系分布、月度趋势、高频词云、时段热力图 | |
| **好友列表** | 搜索、排序、改关系备注；AI 一键推断这位好友的关系和职务 | |
| **关系网络图** | 以"我"为中心的气泡关系图，看往来最多的好友 | |
| **好友详情** | 消息数、发言占比、活跃时段、聊天样本，还有 **AI 情绪分析** 和 **AI 好友画像**（身份/家庭/感情/生活 + 投资偏好） | |
| **年度报告海报** | 一页式海报，可存相册分享；AI 生成年度文案和情绪总结 | |

> 参照物里有的、年轮暂时**还没做**的：情侣/群聊专门分析、MBTI 人格预测、AI 情话生成器这类"娱乐玩法"。这些要不要做，正想听您的意见（见下一节）。

## 三、最想请您回答的几个问题

1. **整体感觉**：现在这个小程序，和您当初在小红书看到的那款相比，达到您预期了吗？哪里超出预期、哪里还差点意思？
2. **最重要的功能**：如果只能保留 3 个功能，您会留哪 3 个？
3. **娱乐玩法要不要**：那款主打**情侣分析、MBTI 人格预测、AI 情话生成器、520 礼物**这类话题玩法。您觉得您的用户会喜欢这些吗？值不值得我补上？
4. **群聊分析**：那款可以分析微信群，年轮目前只做一对一好友。您需要群聊分析吗？
5. **报告长什么样**：那款是"英文研究报告"风、信息很密；年轮是"一页海报"风、适合发朋友圈。您更想要哪种？
6. **隐私**：您和您的用户，**能接受把聊天记录上传到服务器吗**？还是"绝不出手机"是硬要求？
7. **给谁用 / 怎么用**：这个小程序最终是**自己用、送给用户当礼物、还是要做成对外产品收费**？（会决定我后面往哪个方向做）
